## 📱 À propos du projet

**StockChef** est une application mobile en cours de développement qui vous aide à :
- 📸 **Scanner vos produits** et suivre leurs dates de péremption
- 🚨 **Recevoir des alertes intelligentes** avant expiration
- 🧠 **Générer des recettes personnalisées** avec vos restes
- 🛒 **Synchroniser vos courses** avec les drives français

**Site de présentation** : [stockchef.fr](http://www.stockchef.fr/)

## ✨ Fonctionnalités clés

| Fonctionnalité | Description |
|----------------|-------------|
| Scan intelligent | Ajoutez des produits via le code-barres |
| Alertes DLC | Notifications 3 jours avant expiration |
| Recettes IA | Suggestions culinaires avec ChatGPT |
| Synchronisation Drive | Intégration avec Carrefour, Leclerc... |
